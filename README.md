# employee-management
Simple Employee Management Application

## Tech Stack
- Spring Boot
- Spring Data JPA
- PostgreSQL
- IntelliJ IDEA
- Visual Studio Code IDE
- Maven
- Postman
- React JS
- Vite JS
- Bootstrap CSS
- JavaScript
- NPM
- Axios

## Run Application
- **React Application -->** ``` npm run dev ```
- **Spring Boot Application -->** ``` mvn spring-boot:run ```
