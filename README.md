# employee-management
Simple Employee Management Application

## Tech Stack
- Spring Boot
- Spring Data JPA
- PostgreSQL
- IntelliJ IDEA
- Visual Studio Code IDE
- Maven
- Postman
- React JS
- Vite JS
- Bootstrap CSS
- JavaScript
- NPM
- Axios

## Frontend - React
### Run on local
```
npm run dev 
```
The application will be available at ```http://localhost:3000```

## Backend - Spring Boot
### Install
```
mvn clean package
```
### Run on local
```
mvn spring-boot:run
```
The application will be available at ```http://localhost:8080```

## Spring Boot Security
- GET: Admin, User
- ADD: Admin, User
- UPDATE: Amin
- DELETE: Admin
